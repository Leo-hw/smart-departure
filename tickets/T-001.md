**상태**: 🟢 DONE

## 목표

Railway에서 실행 가능한 Python 프로젝트 뼈대를 구성한다.
환경변수 로드, 설정 파일 파싱, main.py 진입점까지 동작 확인.

---

## Acceptance Criteria

- [x] `main.py` 실행 시 환경변수 누락 여부를 체크하고 명확한 에러 메시지 출력
- [x] `shared/config/settings.yaml` 로드 성공
- [x] `requirements.txt` 작성 완료 (google-auth, google-api-python-client, requests, pyyaml)
- [x] `.gitignore`에 `.env`, `.runtime/`, `*.pyc`, `.venv/` 포함
- [x] `README.md`에 로컬 실행 방법 작성
- [x] Railway용 `Procfile` 또는 `railway.toml` 작성 (cron worker 형태)

### 필수 환경변수 목록 (main.py에서 체크)
```python
REQUIRED_ENV_VARS = [
    "GOOGLE_SERVICE_ACCOUNT_JSON",
    "GOOGLE_CALENDAR_IDS",
    "GOOGLE_MAPS_API_KEY",
    "TELEGRAM_BOT_TOKEN",
    "TELEGRAM_CHAT_ID",
    "HOME_ADDRESS",
]
```

### settings.yaml 초기값
SPEC.md §4 참고. `HOME_ADDRESS`는 환경변수로 오버라이드.

### Procfile (Railway cron worker)
```
worker: python main.py
```

Railway에서 cron 스케줄은 대시보드에서 설정 (매 시간 정각: `0 * * * *`).
