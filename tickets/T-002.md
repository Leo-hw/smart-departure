## 구현 상세

### 반환 타입
```python
@dataclass
class CalendarEvent:
    summary: str        # 일정 제목
    location: str       # 장소 (빈 문자열이면 제외됨)
    start_time: datetime  # KST 기준
    event_id: str       # dedup용 고유 ID
    transport_override: str | None  # 일정 설명란에 "transport: driving" 있으면 파싱
```

### transport_override 파싱
일정 description에 아래 형식이 있으면 파싱:
```
transport: driving
```
허용값: `transit`, `driving`, `walking`. 그 외는 None으로 처리.

### Google API 인증
`GOOGLE_SERVICE_ACCOUNT_JSON` 환경변수에서 JSON 파싱 → 서비스 계정 인증  
스코프: `https://www.googleapis.com/auth/calendar.readonly`
