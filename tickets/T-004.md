## 구현 상세

### 권장 반환 타입
```python
@dataclass
class DepartureDecision:
    event: CalendarEvent
    transport_mode: str
    travel_minutes: int
    buffer_minutes: int
    departure_time: datetime
    should_alert: bool
    is_estimated: bool
    provider: str
```

### 알림 판단 규칙
현재 시각이 `departure_time ± alert_window_minutes` 범위 안에 있으면 `should_alert=True`

예:
- 출발 시각이 14:15
- `alert_window_minutes=10`
- 현재 시각이 14:05 ~ 14:25 사이면 알림 대상

### 주의사항
- dedup 판단은 T-006 범위이므로 여기서는 하지 않음
- 알림 전송은 T-005 범위이므로 여기서는 계산 결과만 반환
