## 구현 상세

### 인터페이스
```python
@dataclass
class TravelResult:
    duration_minutes: int
    is_estimated: bool   # fallback 사용 시 True
    provider: str        # "google" or "kakao"

def get_travel_time(origin: str, destination: str, transport_mode: str) -> TravelResult:
    provider = _select_provider(transport_mode)
    return provider.calculate(origin, destination)
```

### provider 선택 규칙 (현재 MVP)
| transport_mode | provider |
|----------------|----------|
| transit | google |
| driving | google (T-007에서 kakao로 교체 예정) |
| walking | google |

### Google Maps Directions API
- endpoint: `https://maps.googleapis.com/maps/api/directions/json`
- 파라미터: `origin`, `destination`, `mode=transit`, `key=GOOGLE_MAPS_API_KEY`, `language=ko`, `departure_time=now`
- 이동 시간 추출: `routes[0].legs[0].duration.value` (초) → 분으로 변환

### 캐시 구조 (.runtime/travel_cache.json)
```json
{
  "origin|destination|transit": {
    "duration_minutes": 35,
    "cached_at": "2026-04-15T10:00:00+09:00"
  }
}
```
