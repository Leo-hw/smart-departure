> ⚠️ T-010과 `scheduler.py`·`settings.yaml`을 함께 수정하고, T-010의 catch-up "준비단계 가장 최근만"
> 로직이 본 티켓의 다단계 prep 위에서 동작해야 함. **T-010 머지 후 시작.** 병렬 금지.

---

## 목표

일정 제목 키워드로 occasion을 분류하고, occasion별로 정의된 준비 단계(stage)마다 알림을 전송한다.
기존 단일 `prep_minutes` 방식을 대체한다.

---

## Acceptance Criteria

- [ ] `shared/config/settings.yaml`에 `prep_profiles` 섹션 추가 (5개 프로필)
- [ ] `core/scheduler.py`에 occasion 분류 로직 추가 — 일정 제목 키워드 → profile 선택, 매칭 없으면 `casual` fallback
- [ ] 각 stage의 알림 시각 = 출발 시각 - `minutes_before_departure`
- [ ] stage별 alert가 dedup 키로 구분되어 각각 독립 전송 (`{event_id}_{stage_label}`)
- [ ] `schedule_today.json`에 stage 목록 포함
- [ ] 기존 단일 `prep_minutes` 설정은 제거
- [ ] 단위 테스트: occasion 분류 + stage 시각 계산 케이스

## 구현 상세

### prep_profiles 구조 (settings.yaml)

```yaml
prep_profiles:
  formal:
    keywords: ["면접", "인터뷰", "결혼식", "웨딩", "장례", "발표", "프레젠테이션", "비즈니스 미팅"]
    stages:
      - label: "샤워 & 준비 시작"
        minutes_before_departure: 120
      - label: "옷 입기 (정장)"
        minutes_before_departure: 45
      - label: "출발"
        minutes_before_departure: 0

  semi_formal:
    keywords: ["소개팅", "데이트", "네트워킹", "커피챗", "동창회", "동문회", "공연", "콘서트", "전시"]
    stages:
      - label: "씻고 준비 시작"
        minutes_before_departure: 90
      - label: "옷 입기"
        minutes_before_departure: 30
      - label: "출발"
        minutes_before_departure: 0

  casual:
    keywords: []   # fallback — 매칭 없을 때 기본값
    stages:
      - label: "준비 시작"
        minutes_before_departure: 60
      - label: "출발"
        minutes_before_departure: 0

  travel:
    keywords: ["여행", "공항", "출국", "캠핑", "글램핑"]
    stages:
      - label: "짐 챙기기 시작"
        minutes_before_departure: 180
      - label: "최종 짐 확인"
        minutes_before_departure: 30
      - label: "출발"
        minutes_before_departure: 0

  sports:
    keywords: ["운동", "헬스", "필라테스", "요가", "등산", "축구", "농구", "테니스"]

      - label: "운동복 & 장비 챙기기"
```

### occasion 분류 로직
- 일정 `summary`에 keywords 중 하나라도 포함되면 해당 profile 선택
- 복수 매칭 시 리스트 상단 profile 우선 (formal > semi_formal > travel > sports > casual)
- 캘린더 description의 `transport: driving` 오버라이드와 동일하게 `occasion: formal` 오버라이드도 지원

### 알림 메시지 예시

준비 단계 알림:
```
🔔 준비 시작할 시간이에요!

📅 최종 면접 (오후 3시 30분)
👔 지금: 샤워 & 준비 시작
🚀 출발: 오후 1시 30분 (2시간 후)
```

출발 알림 (기존 포맷 유지):
```
🚀 출발할 시간이에요!

📅 최종 면접
📍 장소: 강남구 테헤란로 OO빌딩
🕐 시작: 오후 3시 30분
🚌 이동: 대중교통 약 35분
⏰ 출발: 지금!
```
