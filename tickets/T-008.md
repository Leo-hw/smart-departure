# T-008 — 카카오 Maps 자동차 길찾기 연동

**상태**: ⬜ BACKLOG  
**담당**: Codex  
**의존**: T-007

---

## 목표

`transport_mode="driving"` 시 카카오 Maps Directions API를 사용하도록 `maps_service.py`를 확장한다.
MVP에서 Google Maps로 임시 처리하던 자동차 경로를 카카오로 교체한다.

---

## Acceptance Criteria

- [ ] `maps_service.py`의 `_select_provider()` 에서 `driving` → 카카오 provider 반환
- [ ] 카카오 Directions API (`https://apis-navi.kakaomobility.com/v1/directions`) 연동
- [ ] 환경변수 `KAKAO_REST_API_KEY` 사용
- [ ] 반환값 형식은 기존 `TravelResult`와 동일 (`duration_minutes`, `is_estimated`, `provider`)
- [ ] API 실패 시 fallback: 캐시 → 기본값 30분 (기존 정책 동일)
- [ ] 단위 테스트: `tests/test_maps_service.py`에 driving 케이스 추가

---

## 구현 상세

### 카카오 Directions API
- endpoint: `https://apis-navi.kakaomobility.com/v1/directions`
- 헤더: `Authorization: KakaoAK {KAKAO_REST_API_KEY}`
- 파라미터: `origin={lng},{lat}`, `destination={lng},{lat}`, `priority=RECOMMEND`
- 주소 → 좌표 변환: 카카오 로컬 API (`https://dapi.kakao.com/v2/local/search/address.json`)
- 이동 시간 추출: `routes[0].summary.duration` (초) → 분으로 변환

### provider 선택 규칙 (T-008 이후)
| transport_mode | provider |
|----------------|----------|
| transit | google |
| driving | kakao |
| walking | google |
